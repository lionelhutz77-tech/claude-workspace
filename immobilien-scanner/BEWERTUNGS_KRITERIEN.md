# 📊 Bewertungs-Kriterien (investmentpunk-basiert)

## Der 60-Sekunden-Filter (Was wird gescannt)

✅ **Bruttorendite > 6%** → Weiter  
✅ **Vollvermietung oder realistisches Mietpotenzial** → Weiter  
✅ **B-Lage (Oberhausen: zentral/Sterkrade)** → Weiter  
✅ **Finanzierbar mit 70-100k EK (80% LTV)** → KAUFEN  

**Eliminiert 80% der schlechten Fälle bevor du einen Cent ausgibst!**

---

## Die 4 Kernmetriken (Was Groq berechnet)

### 1️⃣ **Bruttorendite** (Annual Gross Yield)
```
Bruttorendite = (Jahresmiete / Kaufpreis) × 100
```

**Bewertung:**
| Brutto | Bewertung | Aktion |
|--------|-----------|--------|
| < 6% | ❌ Break-even | Skip |
| 6-7% | ⚠️ Minimal akzeptabel | Prüfen |
| 6-8% | ✅ Detailprüfung | Weitermachen |
| > 8% | 🔥 Buy-Signal | Zuschlagen |

**Warum 6%?**
- ~4% für Finanzierungskosten (Zins + Tilgung)
- ~1% für Instandhaltung
- ~1% für Leerstand
- = 0% Netto (gerade break-even)

### 2️⃣ **Cashflow vor Kredit** (Monthly Operating Surplus)
```
Cashflow = (Kaltmiete × Wohnungen) - (Verwaltung + Instandhaltung + Versicherung + Leerstand)
```

**Beispiel (4 Wohnungen à 950€):**
- Miete: 3.800€
- Verwaltung (10%): -380€
- Instandhaltung (1,50€/m²): -150€ (bei ~100m² pro Whg)
- Versicherung: -40€
- Leerstand (3%): -114€
- **= 3.116€ monatlich**

### 3️⃣ **Netto-Cashflow** (Cash to Owner)
```
Netto-Cashflow = Cashflow - Kreditrate
```

**Kreditrate (Beispiel):**
- Kaufpreis: 450.000€
- LTV 80% = Fremdkapital 360.000€
- 4% Zinssatz, 1% Tilgung = 5% p.a.
- 30 Jahre Laufzeit
- Monatliche Rate: ~1.900€

**Netto = 3.116€ - 1.900€ = 1.216€/Monat**

**Bewertung:**
- < 500€: ⚠️ Knapp, hohes Risiko
- 500-1.000€: ✅ OK
- > 1.000€: 🔥 Sehr gut!

### 4️⃣ **Netto-Rendite auf Eigenkapital** (Return on Equity)
```
Netto-Rendite = (Netto-Cashflow × 12 / Eigenkapital) × 100
```

**Beispiel:**
- Eigenkapital: 90.000€ (20% von 450k)
- Netto-Cashflow: 1.216€/Monat = 14.592€/Jahr
- Netto-Rendite: (14.592 / 90.000) × 100 = **16,2%**

**Bewertung:**
- 5-8%: ⚠️ Eher schwach
- 8-12%: ✅ Gut
- > 12%: 🔥 Sehr gut (Hebel-Effekt funktioniert!)

## Die Roten Flaggen (Was ist ein KO-Kriterium?)

### ❌ **AUSSCHLUSS (Score < 40):**
1. **Bruttorendite < 6%** ← Break-even, kein Spielraum
2. **Netto-Cashflow < 500€/Monat** ← Zu knapp, hohes Risiko
3. **Renovierungskosten > 50.000€** ← Zu teuer, verschlechtert Rendite
4. **C-Lage mit hohem Leerstand-Risiko** ← Theorie vs. Realität

### ⚠️ **WARNUNG (Score 40-60, grün prüfen):**
- **Baujahr < 1950** ← Wahrscheinlich feuchte Keller, alte Heizung
- **Renovierungskosten 30-50k€** ← Substanziell, aber verkraftbar
- **Große Wohnungen einzeln** ← investmentpunk rät ab ("Never buy as beginner")
- **Neue Gebäude < 4% Brutto** ← Makler kassiert, du zahlst drauf

### ✅ **GRÜNES LICHT (Score 60+):**
- **6-8% Bruttorendite + stabile Lage** = Detailprüfung starten
- **> 8% Brutto in B-Lage** = Zuschlagen!
- **Baujahr > 1970 + Renovierungen recent** = Guter Zustand
- **MFH mit Garten, Garage, Balkon** = Mietpotenzial hoch

## Wie Groq das Score-System nutzt

**Base-Score: Bruttorendite**
- 6% = 0 Punkte (minimum)
- 7% = 25 Punkte
- 8% = 50 Punkte
- 10% = 100 Punkte

**+ Boni:**
- +10 Punkte: > 8% Brutto
- +10 Punkte: Netto-Cashflow > 1.000€
- +10 Punkte: Baujahr > 1970
- +5 Punkte each: Garten, Balkon, Garage, Renoviert

**- Mali:**
- -20 Punkte: Baujahr < 1950
- -15 Punkte: Renovierungen 30-50k€
- -25 Punkte: Renovierungen > 50k€

**→ Final Score = Base + Boni - Mali (min 0, max 100)**

## Was investmentpunk NICHT kauft

### ❌ C-Lagen-Ruinen
- Papier: 6% Brutto ✓
- Realität: 40% Leerstand ✗
- Netto-Rendite: Negativ
- → **Rendite-Friedhof**

### ❌ Neubau-Glanz < 4% Brutto
- Der Makler kassiert Provision
- Du zahlst jeden Monat drauf
- → **Schneeballsystem für Makler**

### ❌ "Schnäppchen" ohne Mietpotenzial
- Kein Mieterwachstum
- Kein Cashflow
- Egal wie billig
- → **Teure Pille**

## Oberhausen Spezifika

**Lage-Kategorie: B-Lage mit High-Yield Potential**

✅ **Vorteile:**
- Zentrale Lage (Sterkrade, Innenstadt)
- Affordable purchase prices (< A-Lagen)
- Stabile Mietennachfrage
- 6-8% Bruttorendite realistisch

⚠️ **Risiken:**
- Above-average vacancy (C-Lagen-Effekt in schlechten Stadtteilen)
- Strukturwandel vom Stahl → Service/Retail
- Bevölkerungswachstum nicht überall

✅ **Filterung im Scanner:**
- Zentrale Bezirke: ✅ (Sterkrade, Innenstadt)
- Randbezirke: ⚠️ (Leerstand-Risiko höher)
- Price-Factor < 18: ✅ (Preis-Miete ok)
