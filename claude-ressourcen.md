# Kostenlose Claude-Ressourcen

- **Awesome Claude Skills** (kuratierte Liste): https://github.com/ComposioHQ/awesome-claude-skills

## Hinweis
Skills/Plugins vor Installation kurz prüfen (Quelle, Security-Audit). Wir bauen eigene Skills bereits direkt im Workspace unter `.claude/skills/` — externe nur ergänzend nutzen, wo sie echten Mehrwert bringen.
